**[1] SnipGen — CRISPR guide RNA design platform**
*Python · Flask · scikit-learn · GitHub Actions · Render*

> ML on-target scoring model trained on 5,310 real CRISPR screen guides from the Doench 2016 / Azimuth dataset. Full Pytest suite integrated into CI/CD pipeline. **19 production deployments.**

**[2] MedAssist Edge — offline RAG diagnostic assistant**
*Python · Ollama · LangChain · ChromaDB · Streamlit*

> Local RAG system for medical device troubleshooting. Initial corpus audit found **6 of 8 indexed sources contained zero troubleshooting value** (marketing brochures, a press release, the project's own spec document). System was rebuilt from scratch: corpus replaced with FDA MAUDE failure reports, IAEA QA handbooks, AAPM protocols, WHO maintenance workbooks, and NHS/HSE regulatory guidance. Distance threshold filter at **0.85** drops low-confidence retrieval before LLM generation. Structured DIAGNOSIS / STEPS / EVIDENCE / GAPS output on every query. Isolation Forest anomaly detection validated at **100% precision, 56% recall** at 5% contamination with randomised injection windows.

→ [github.com/ldharwal-asu/MedAssist](https://github.com/ldharwal-asu/MedAssist)
